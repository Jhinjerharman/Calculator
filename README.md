🧮 Simple Python Calculator

A beginner-friendly calculator built using Python functions. This project demonstrates basic programming concepts like functions, conditionals, and user input.

🚀 Features

✨ Perform basic arithmetic operations:

➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division

🛡️ Handles division by zero errors
📚 Clean and easy-to-understand code
👨‍💻 Perfect for beginners learning Python

🛠️ Technologies Used
Python 🐍
📂 Project Structure
calculator.py   # Main Python file containing the calculator code

Run the program:

python calculator.py
💡 How It Works
The program asks the user to choose an operation.
It takes two numbers as input.
Based on the selected option, it calls the corresponding function:
add()
subtract()
multiply()
divide()
Displays the result.
